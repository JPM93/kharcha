# 🦚 Kharcha — Personal Budget & Expense Tracker

Kharcha is a clean and modern personal expense tracking web app built using pure HTML, CSS, and JavaScript.

It helps manage:

* 🛒 Shopping expenses
* 💡 Utility bills
* 🏦 EMI tracking
* 💚 Savings & transfers
* 📊 Spending reports and analytics

## ✨ Features

* Monthly budget management
* Category-based shopping lists
* Item status tracking (Pending / Bought / Cancelled)
* EMI and bill reminders
* Savings tracking
* Expense analytics & reports
* Cross-month recurring item insights
* Import / Export backup support
* Fully responsive premium UI
* Works completely offline using LocalStorage

## 🛠 Tech Stack

* HTML5
* CSS3
* Vanilla JavaScript
* LocalStorage API

## 📦 Storage

All data is stored locally in the browser using LocalStorage.
No backend or account is required.
